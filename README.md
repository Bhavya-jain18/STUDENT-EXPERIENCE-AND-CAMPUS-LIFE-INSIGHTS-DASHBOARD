# Student Experience and Campus Life Insights Dashboard

This project is a Power BI dashboard created to analyze student life, stress levels, and study habits.

## Tools Used
- Power BI
- Data Visualization
- Survey Data Analysis

## Dashboard Insights
- Total Students: 2998
- Average Stress Level: 3.04
- Average Well-being: 3

## Key Analysis
- Gender distribution of students
- Library visit patterns
- Study hours distribution
- Daily phone usage
- Academic stress levels
- Student suggestions for campus improvements
